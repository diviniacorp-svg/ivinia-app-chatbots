# Skill: Guión Técnico de Reel para DIVINIA

Genera el guión técnico completo de un Reel de Instagram para DIVINIA: estructura visual frame a frame, voz en off, texto on screen, prompts Freepik y notas de producción.

---

## CONTEXTO DE PRODUCCIÓN

**Duración target:** 15-30 segundos (sweet spot para alcance)
**Formato:** 9:16 vertical (1080x1920px)
**Plataforma principal:** Instagram Reels (luego TikTok y YouTube Shorts)
**Herramientas visuales disponibles:**
- **Freepik Mystic** → imágenes estáticas (fondo, elementos)
- **Freepik Seedance** → texto a video animado
- **Freepik Kling** → video cinemático realista
- **Remotion** → animaciones programáticas (gráficos, stats, UI)

---

## FÓRMULA DE REEL DIVINIA

Cada reel sigue esta estructura:

```
[0-3s]  HOOK   → Para el scroll. Pregunta, dato shock, declaración polémica
[3-20s] BODY   → Desarrolla la promesa. Muestra el producto. Número concreto.
[20-30s] CTA   → Acción específica. "Mandá TURNO al WhatsApp". Link en bio.
```

**Hooks que funcionan por tipo:**

| Tipo hook | Ejemplo |
|---|---|
| Pregunta dolorosa | "¿Cuántas horas perdés por semana dando turnos por WA?" |
| Dato shock | "El 73% de los turnos se cancelan cuando no hay recordatorio automático" |
| Comparación | "Mientras vos contestás WA manualmente, tu competencia ya usa IA" |
| Declaración polémica | "Contraté una empleada que trabaja 24hs y cobra $0" |
| Transformación | "Así era mi agenda antes. Así es ahora con DIVINIA." |

## Paso 1 — Definir el reel

Preguntale a Joaco:
1. **Tema / ángulo:** (ej: "costo de dar turnos manualmente", "antes vs después", "demo en vivo")
2. **Producto protagonista:** Turnero / Landing / NUCLEUS / DIVINIA general
3. **Segmento objetivo:** peluquería-estética / clínica-médico / PYME general
4. **¿Hay material real disponible?** pantallas reales / capturas / cero material (solo IA)

Si no hay contexto, generá un reel de **Turnero IA para peluquerías, estilo demo "antes/después"**.
