# Skill: Generar Post de Instagram para DIVINIA

Genera un post completo para @divinia.ia: brief visual, caption lista para publicar, hashtags y prompt Freepik para la imagen.

---

## SISTEMA DE MARCA (siempre aplicar)

**Paleta:**
- Negro DIVINIA: `#0A0A0A` — fondos, peso, autoridad
- Violeta IA: `#8B5CF6` — acento principal
- Violeta suave: `#A78BFA` — gradientes
- Blanco violeta: `#F5F0FF` — texto sobre oscuro
- Rosa impacto: `#EC4899` — CTAs, urgencia
- Verde resultado: `#10B981` — métricas, éxito
- Gradiente signature: `linear-gradient(135deg, #8B5CF6 0%, #EC4899 100%)`

**Tipografía:**
- Display (títulos): Clash Display Bold
- Body: Inter Medium
- Stats/datos: Space Grotesk Bold

**Tono de voz — REGLAS DURAS:**
- Siempre en vos/sos/tenés (argentino)
- NUNCA: "soluciones innovadoras", "plataforma robusta", "estado del arte"
- SÍ: precio directo, resultado concreto, beneficio específico
- CTA siempre específica: "Mandá TURNO al WhatsApp" — nunca "contáctanos"
- Emoji estratégico: máximo 4 por caption

---

## Paso 1 — Definir el post

Preguntale a Joaco (o inferí del contexto):
1. **¿Qué tipo de post?** → educativo / demo / social proof / viral / motivacional
2. **¿Qué producto muestra?** → Turnero IA / Landing Page / NUCLEUS IA / DIVINIA general
3. **¿A quién apunta?** → peluquería-estética / restaurante / clínica / PYME general
4. **¿Hay un ángulo o tema específico?** (ej: "costo de perder turnos", "empleado digital", etc.)

Si no hay contexto, generá un post **educativo sobre Turnero IA para peluquerías/estética** (segmento hero).

---

## Paso 2 — Elegir el pilar

| Pilar | % objetivo | Objetivo | Ejemplos de hook |
|---|---|---|---|
| Educativo | 30% | Awareness + confianza | "¿Sabías que...?", "El error que cometen..." |
| Demo | 25% | Conversión directa | "Mirá cómo funciona...", "En 2 minutos..." |
| Social proof | 20% | Credibilidad | "Esto le pasó a [nombre]...", "Resultado real..." |
| Viral | 15% | Alcance nuevo | Comparaciones, "mientras vos...", listas |
| Motivacional | 10% | Fidelización | "Si sos dueño de negocio...", reflexiones |

---

## Paso 3 — Generar el brief visual

Producí el brief exacto para diseñar la imagen:

```
📐 BRIEF VISUAL

Formato: [Feed 1080x1080 / Reel cover 1080x1920]
Fondo: Negro #0A0A0A (sólido) o gradiente oscuro violeta-negro
Elemento principal: [mockup de phone / número grande / comparación / persona]
Texto overlay: [máx. 7 palabras, Clash Display, blanco #F5F0FF o violeta #A78BFA]
Acento: borde o detalle en gradiente violeta-rosa
Logo: DIVINIA esquina inferior derecha, blanco 30% opacidad
```

---

## Paso 4 — Generar el caption

Estructura:

```
[HOOK — primera línea, tiene que parar el scroll]

[DESARROLLO — 3-5 líneas, específico, con número o resultado concreto]

[CTA directa — 1 línea máxima]

[Hashtags]
```

**Reglas:**
- Primera línea = el hook (lo más importante, visible sin expandir)
- Incluir al menos 1 número concreto o resultado
- CTA siempre apunta a WhatsApp o link en bio — nunca a nada genérico
- Longitud: 80-150 palabras (ideal feed), puede ser más corto para reel

---

## Paso 5 — Generar prompts Freepik

Producir **3 prompts** en inglés:

**A) Imagen estática para Mystic (thumbnail / feed):**
```
[Descripción detallada: elementos, colores, estilo, formato]
Style: product photography, dark background, purple and violet accents, 
professional Argentine business aesthetic, 1080x1080px, high quality
```

**B) Video corto para Seedance (animación texto-a-video):**
```
[Descripción de la escena animada]
Purple and black color scheme, smooth animation, vertical 9:16 format,
modern UI mockup on smartphone, Argentine business context, 4K quality
```

**C) Variante alternativa para A/B test:**
```
[Descripción alternativa con diferente ángulo visual]
```
